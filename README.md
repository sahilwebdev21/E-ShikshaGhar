# 📚 E-Shikshaghar  

E-Shikshaghar is a dynamic **e-learning platform** that enables students and teachers to interact, learn, and engage in online education. Built with a modern tech stack, this platform provides AI-powered course recommendations, real-time chat, and gamification to enhance the learning experience.

---

## 🚀 Tech Stack  

- **Frontend:** Next.js (React framework)  
- **Backend:** Python (Django)  
- **Database:** MySQL (XAMPP for local development)  

---

## 🔥 Key Features  

✔ **AI-Powered Course Recommendations**  
✔ **Live Chat for Teacher-Student Interaction**  
✔ **Gamification (Badges & Leaderboards)**  
✔ **Responsive & Dynamic UI**  
✔ **Secure User Authentication**  
✔ **Course Management & Progress Tracking**

## 🛠 Installation & Setup  

### 1️⃣ Clone the Repository  
```sh
git clone https://github.com/your-username/e-shikshaghar.git
cd e-shikshaghar
```

### 2️⃣ Install Frontend Dependencies  
```sh
cd frontend
npm install
npm run dev  # Starts the Next.js development server
```

### 3️⃣ Setup Backend (Django)  
```sh
cd backend
pip install -r requirements.txt  # Install Python dependencies
python manage.py migrate  # Apply database migrations
python manage.py runserver  # Run Django server
```

### 4️⃣ Configure MySQL Database (XAMPP)  
- Start **Apache** and **MySQL** in XAMPP Control Panel  
- Create a database named **e_shikshaghar** in phpMyAdmin  
- Update the database credentials in **backend/settings.py**
